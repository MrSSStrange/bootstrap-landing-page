This is my first learning project built with HTML and Bootstrap.

## About the project

A responsive landing page for a fictional moving company.

The goal of this project was to practice:
- Using Bootstrap components
- Page layout and structure
- Responsive design principles
- Working with Bootstrap via CDN
- Implementing Bootstrap dark theme

## Features

- Responsive navigation bar
- Hero section with call-to-action buttons
- Feature section with icons
- Image carousel
- Footer with multiple columns
- Dark theme enabled using Bootstrap 5 (`data-bs-theme="dark"`)

## Technologies

- HTML5
- Bootstrap 5
- CSS

## What I learned

- How to use Bootstrap grid system
- How to work with ready-made Bootstrap components
- How to build a simple landing page layout
- How to enable and use Bootstrap dark mode
- How to structure a small frontend project
